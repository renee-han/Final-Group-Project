# Final-Group-Project: Renee Han and Amelia Smith

##Project Description 
[Our project is a web application that allows users to input a location and receive activity recommendations based on the current weather. It offers a variety of activities from indoor, to outdoor, to free, and family-friendly. It uses the OpenAI API, Mapbox API, and OpenWeather API to provide users with an easy itinerary for planning their day.]

##Setup Instruction 
[For this project, you will need to have Python installed on your machine. You will also need to install the required libraries, which can be done using pip. The main libraries we are using are Flask for the web application, requests for making API calls, and dotenv for managing environment variables. You can install these libraries by running the following command in your terminal: `pip install Flask requests python-dotenv`. Additionally, you will need to set up API keys for the OpenAI API, Mapbox API, and OpenWeather API. These keys should be stored in a .env file in the root directory of your project for security purposes.]

##Usage
[To use the application, the user will start on the homepage and enter a location in the search bar. The app will then display the current weather for that location along with a list of recommended activities that are suitable for the weather conditions. Users can click on links for each activity to get more details on the location's webpage, such as location, hours of operation, and reviews.]
